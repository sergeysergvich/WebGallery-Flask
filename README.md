# WebGallery-Flask

Учебное веб-приложение для просмотра и управления изображениями,
реализованное на языке Python с использованием микрофреймворка Flask.

Проект разработан в рамках курсовой работы по дисциплине
«Объектно-ориентированное программирование».

## Функциональные возможности
- загрузка изображений через веб-интерфейс
- автоматическое отображение изображений из папок
- добавление изображений в избранное
- хранение данных без СУБД (JSON + файловая система)
- адаптивный пользовательский интерфейс

## Технологический стек
- Python
- Flask
- HTML5
- CSS3
- Jinja2
